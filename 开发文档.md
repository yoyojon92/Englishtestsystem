## 1. 项目概述

### 1.1 项目背景

英语学习诊断系统是一套面向 KET/PET 考生的智能学习平台，通过 L1-L5 五层架构实现从数据采集到学习规划的完整闭环。

### 1.2 核心价值

| 层级 | 名称 | 功能 |
|------|------|------|
| **L1** | 数据采集 | 记录答题行为、时间、正确性 |
| **L2** | 题库系统 | KET/PET 真题管理 |
| **L3** | 认知诊断 | CEFR 等级计算 |
| **L4** | 能力画像 | 技能得分、弱项分析 |
| **L5** | 学习规划 | 个性化课程推荐 |

### 1.3 目标用户

- KET/PET 考生及家长
- 英语培训教师
- 教育机构管理者

## 6. API 文档

### 6.1 测试相关

#### 创建测试会话
```
POST /api/v1/test/session/create
Body: { channel: string, examType: "KET" | "PET", count?: number }
Response: { sessionId: string, questions: Question[] }
```

#### 提交答案
```
POST /api/v1/test/session/:id/answer
Body: { questionId: string, answer: string, timeSpent: number }
Response: { success: boolean }
```

#### 提交测试
```
POST /api/v1/test/session/:id/submit
Body: {}
Response: { reportId: string, cefrLevel: string, score: number }
```

#### 获取报告
```
GET /api/v1/test/report/:id
Response: { report: Report }
```

### 6.2 分享追踪

#### 生成分享链接
```
POST /api/v1/share/generate
Body: { channel: string, campaign?: string }
Response: { shareCode: string, shareUrl: string }
```

#### 追踪点击
```
POST /api/v1/share/track
Body: { shareCode: string }
Response: { success: boolean }
```

#### 渠道统计
```
GET /api/v1/share/stats
Response: { channels: ChannelStats[] }
```

#### 获取题目列表
```

```

## 7. 数据库设计

### 7.1 核心表结构

#### test_sessions
```sql
CREATE TABLE test_sessions (
  id VARCHAR(50) PRIMARY KEY,
  student_id VARCHAR(50),
  exam_type VARCHAR(10) NOT NULL,
  channel VARCHAR(50),
  status VARCHAR(20) DEFAULT 'active',
  started_at TIMESTAMP DEFAULT NOW(),
  completed_at TIMESTAMP,
  answers JSONB DEFAULT '{}'
);
```

#### test_reports
```sql
CREATE TABLE test_reports (
  id VARCHAR(50) PRIMARY KEY,
  session_id VARCHAR(50) REFERENCES test_sessions(id),
  student_id VARCHAR(50),
  cefr_level VARCHAR(10),
  score INTEGER,
  skill_scores JSONB,
  weak_points TEXT[],
  recommendations TEXT[],
  created_at TIMESTAMP DEFAULT NOW()
);
```

#### share_links
```sql
CREATE TABLE share_links (
  id VARCHAR(50) PRIMARY KEY,
  share_code VARCHAR(20) UNIQUE NOT NULL,
  channel VARCHAR(50),
  campaign VARCHAR(100),
  click_count INTEGER DEFAULT 0,
  usage_count INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);
```
