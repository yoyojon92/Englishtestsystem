## 1. 家长端操作指南

### 1.1 注册与登录

#### 手机号注册
1. 打开APP，点击「登录/注册」
2. 输入手机号，点击「获取验证码」
3. 输入收到的6位验证码
4. 设置密码（可选），完成注册

#### 微信一键登录
1. 点击「微信登录」按钮
2. 授权微信账号登录
3. 首次登录自动创建账号

### 1.2 给孩子做水平测试

#### 入口位置
首页 → 「📝 水平测试」卡片

#### 测试流程
1. **阅读测试说明**
   - 测试时长：15分钟
   - 测试题数：25道
   - 涵盖：听力、阅读、写作、口语

2. **开始测试**
   - 点击「开始测试」按钮
   - 页面显示倒计时和进度条

3. **答题**
   - 选择题：点击选项即可
   - 匹配题：拖拽或点击匹配
   - 填空题：输入答案

4. **提交**
   - 做完所有题后点击「提交」
   - 或等待15分钟倒计时结束自动提交

> 📸 [截图位置：测试页面，展示倒计时和题目]

### 1.3 查看测试报告

#### 报告入口
测试完成后自动跳转到报告页面，或在「我的」→「测试记录」中查看

#### 报告内容
1. **CEFR能力等级**
   - 整体等级（A1-C2）
   - 听说读写四项分项等级

2. **能力雷达图**
   - 可视化展示各项能力

3. **薄弱项分析**
   - Top 2 需要加强的知识点
   - 推荐学习方向

4. **服务推荐**
   - 线上课/线下课/自学推荐

### 1.4 购买服务

#### 解锁完整报告
1. 测试报告底部点击「解锁完整报告」
2. 选择支付方式（微信/支付宝）
3. 完成支付后查看完整报告

#### 预约体验课
1. 报告页面点击「预约线下体验课」
2. 选择就近校区和时间
3. 填写联系人信息
4. 确认预约

## 3. 常见问题

### 3.1 测试相关

**Q: 测试有时间限制吗？**
A: 测试限时15分钟，超时自动提交。

**Q: 可以重复做测试吗？**
A: 可以，但完整报告需要购买后才能查看历史记录。

**Q: 测试结果准确吗？**
A: 系统基于认知诊断理论，准确率约85%，建议作为参考。

### 3.2 支付相关

**Q: 支付安全吗？**
A: 使用微信/支付宝官方支付接口，安全可靠。

**Q: 可以退款吗？**
A: 详见「服务条款」，7天内未使用可申请退款。

### 3.3 账号相关

**Q: 忘记密码怎么办？**
A: 点击登录页「忘记密码」，通过手机验证码重置。

**Q: 如何解绑微信？**
A: 「我的」→「账号安全」→「解绑微信」
